点击下面的按钮查看不同类型的Toast通知

代码示例：

Toast.success('操作成功！');

// 自定义配置

    type: 'success',
    title: '上传完成',
    message: '文件已成功上传',
    duration: 5000
});

💬 确认对话框

新的确认对话框替代原生confirm，更美观且功能强大

代码示例：

const confirmed = await Confirm.show({
    title: '确认操作',
    message: '确定要执行此操作吗？',
    type: 'warning',
    confirmText: '确定',
    cancelText: '取消'
});

if (confirmed) {
    // 用户点击了确定
    Toast.success('操作已确认');
}
