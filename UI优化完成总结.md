## 🚀 使用方法

### 方法1: 完整引入(推荐)

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>页面标题 - Benchmark Web Runner</title>

    <!-- 引入设计系统 -->
    <link rel="stylesheet" href="/css/main.css">
    <link rel="stylesheet" href="/css/navigation.css">
    <link rel="stylesheet" href="/css/animations.css">
</head>
<body>
    <!-- 使用组件 -->
    <button class="btn btn-primary">按钮</button>
    <div class="card">卡片</div>
</body>
</html>
```

### 方法2: 使用模板

```bash
# 复制模板
cp /public/template.html /public/new-page.html

# 修改内容
# 1. 更新标题
# 2. 修改导航active状态
# 3. 填充实际内容
```

### 方法3: 渐进式迁移

```html
<!-- 第1步: 引入新CSS -->
<link rel="stylesheet" href="/css/main.css">

<!-- 第2步: 旧代码继续工作(向后兼容) -->
<button class="btn-primary">旧按钮</button>

<!-- 第3步: 逐步替换为新类名 -->
<button class="btn btn-primary">新按钮</button>

<!-- 第4步: 删除内联<style>标签 -->
```

## 💡 最佳实践

### ✅ 推荐做法

1. **新页面**: 从 template.html 开始
2. **颜色**: 使用 CSS变量,不要硬编码
3. **间距**: 使用 spacing系列,不要用固定px
4. **组件**: 使用提供的组件类,不要自己写
5. **动画**: 使用预定义动画,注意性能
6. **响应式**: 使用提供的断点类

### ❌ 避免做法

1. **不要** 修改 design-tokens.css 的变量名
2. **不要** 在HTML中写内联样式
3. **不要** 创建重复的组件样式
4. **不要** 过度使用动画
5. **不要** 忽略移动端适配
6. **不要** 使用固定的px值
