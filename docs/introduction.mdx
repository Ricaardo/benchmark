@bilibili-player/benchmark 可以一键式的替你测量网页的运行时性能, 从而帮助你找到性能瓶颈, 优化你的网页.

提供的指标多种多样, 包括但不限于 CPU, 内存, 重排, 重绘等

本模块提供了多种多样的使用方式:
1. 通过 [Fe-PerfCat 平台](./perfcat-reference/home/home.mdx) 进行长期的性能监测
2. 通过 [Fe-PerfCat SDK](./perfcat-reference/open/init.mdx) 进行接口调用
3. 通过 [命令行工具](./benchmark-sdk-reference/get-started.mdx) 可以在本地快速的进行性能测试
4. 作为一个 [独立的模块](./benchmark-sdk-reference/advanced/js-api.mdx), 你可以在你的项目中使用它
