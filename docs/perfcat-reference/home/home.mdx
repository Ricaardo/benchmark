# 🚀 FE-PerfCat 🐱 指北

[FE-PerfCat - Web 性能监测平台](https://fe-perfcat.bilibili.co)

**FE-PerfCat** 是一个 **Web 性能监测平台**，旨在帮助开发者 **精准分析 Web 性能瓶颈**，优化用户体验。

无论是日常项目发版前后的 **CI/CD 集成自动化测试**，还是线上核心页面或项目的 **长期性能健康度监测**，FE-PerfCat 都能提供**一站式的 Web 性能测评及监控能力**，让你的 Web 页面始终保持 **流畅 & 高效**！

### 🚀 FE-PerfCat 🐱 能做什么？

### 🌏 全面的性能监测
FE-PerfCat 通过自动化工具 & 能力，帮助开发者**全方面了解及监控 Web 性能**，并提供完善的量化指标及优化建议。

包括但不限于以下：

✅ **初始化性能**（ Initialization Performance ）：分析页面初始化过程中的性能瓶颈

✅ **运行时性能**（ Runtime Performance ）：分析页面稳定运行过程中的性能瓶颈

✅ **内存泄露检测**（ Memory Leak Analysis ）：分析页面长时间运行过程中的内存泄露问题
