### 1. 新建 Case 时启动本地 Node.js 服务
#### 需要以下环境：
- Node.js 版本 >= 20 | [下载地址](https://nodejs.org/zh-cn/download)
- npm 版本 >= 10 （一般跟随 node 下载）
- iterm2（MacOS下 推荐）| [下载地址](https://iterm2.com/downloads.html) 或 手动复制命令到终端运行

可以在任何终端运行复制命令，包括系统自带的终端、VS Code 终端等
