[Benchmark SDK](https://dejavu.bilibili.co/material/package/@bilibili-player/benchmark) 是 PerfCat 平台使用的性能测试 SDK，提供了一套完整的性能测试能力，包括但不限于以下：
- **初始化性能**（ Initialization Performance ）：分析页面初始化过程中的性能瓶颈
- **运行时性能**（ Runtime Performance ）：分析页面稳定运行过程中的性能瓶颈
- **内存泄露检测**（ Memory Leak Analysis ）：分析页面长时间运行过程中的内存泄露问题

[Benchmark SDK](https://dejavu.bilibili.co/material/package/@bilibili-player/benchmark) 提供了本地命令行 CLI 和 js 调用两种使用方式，满足不同场景的性能测评需求。
